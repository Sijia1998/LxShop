## 基于Vue2.0+MongoDB+Express4.x实现一个商城系统
> 项目简介 ：包含添加购物车，支付等功能的商城系统

+ 项目技术选型（前后端分离开发）
    + 前端：HTML、CSS、Javascript、Vue2.x、Vuex、Vue-Router
    + 后端：Node.js
    + 数据库：MongoDB
    + 服务端部署：CentOS、nginx
